# Galasa Integration Tests
This repository contains the integration tests that are used to test Galasa Managers. Currently, this repository only contains a test for the Docker Manager.

## Documentation

More information can be found on the [Galasa Homepage](https://galasa.dev). Questions related to the usage of Galasa can be posted in the [galasa-users channel](https://openmainframeproject.slack.com/archives/C05ST4K4K54) on the Open Mainframe Project Slack workspace.

## Where can I get the latest release?

Find out how to install Galasa from our [Installing the Galasa CLI](https://galasa.dev/docs/cli-command-reference/installing-cli-tool/) documentation.

Other repositories are available via [GitHub](https://github.com/galasa-dev). 

## Contributing

If you are interested in the development of Galasa, take a look at the documentation and feel free to ask a question in the [galasa-users channel](https://openmainframeproject.slack.com/archives/C05ST4K4K54) on the Open Mainframe Project Slack workspace, or raise new ideas/features/bugs as Issues on [GitHub](https://github.com/galasa-dev/projectmanagement).

Take a look at the [contribution guidelines](https://github.com/galasa-dev/galasa/blob/main/CONTRIBUTING.md).
